# SmoothOps Landing Page

High-fidelity, cinematic "1:1 Pixel Perfect" landing page built for SmoothOps.
The design utilizes the **Preset C: Brutalist Signal (Raw Precision)** aesthetic with a fully functioning interactive architecture.

## Tech Stack
* **Framework:** React 19 + Vite
* **Styling:** Tailwind CSS v3.4.17
* **Animations:** GSAP 3 (with ScrollTrigger)
* **Icons:** Lucide React

## Local Development
To run this project locally:

1. Clone the repository
2. Run \`npm install\` to install dependencies
3. Run \`npm run dev\` to start the local development server

## Production Build
To create a production-ready build:
1. Run \`npm run build\`
2. The optimized output will be available in the \`/dist\` folder.
